# 🚗 Car Gallery Interactive

Car Gallery Interactive is a simple HTML-based web project that showcases a collection of car images. Users can click on any car image to reveal detailed information about that specific model. This project demonstrates the use of **HTML**, **CSS**, and **JavaScript** to build an interactive image gallery with dynamic content display.

---

## ✨ Features
- Clean and responsive car image gallery  
- Click-to-reveal car details (make, model, specs, etc.)  
- Lightweight and beginner-friendly codebase  
- Easy to extend with more cars or additional styling  

---

## 🎯 Purpose
This project is designed as a learning exercise for web development beginners to practice:
- Structuring content with **HTML**  
- Styling with **CSS**  
- Adding interactivity with **JavaScript**
